# gym-subscription-bot

Telegram-бот для управления абонементами фитнес-клуба.

## Возможности

- Покупка абонементов
- Учет посещений
- QR (ID) для доступа
- Уведомления
- Дашборд с аналитикой

## Технологии

- Python
- aiogram
- SQLite
- matplotlib

## Запуск

```bash
pip install -r requirements.txt
python -m bot.main
python dashboard/dashboard.py
