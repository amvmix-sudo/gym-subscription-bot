# gym-subscription-bot

Telegram-бот для управления абонементами фитнес-клуба.

## Возможности

- Покупка абонементов (разовый, недельный, месячный, годовой)
- Учет посещений
- QR (ID) для доступа в зал
- Уведомления о статусе абонемента
- Дашборд с аналитикой посещений

---

## Технологии

- Python
- aiogram
- SQLite
- matplotlib

---

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/amvmix-sudo/gym-subscription-bot.git
cd gym-subscri
```
### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```
### 3. Настройка токена
#### Создай файл .env в корне проекта:

```env
BOT_TOKEN=your_telegram_bot_token
```
#### ⚠️ Токен не хранится в репозитории по соображениям безопасности.

### 4. Запуск бота

```bash
python -m bot.main
```

---

#### Дашборд

```bash
python dashboard/dashboard.py
```
#### Отображает:
- Посещения по дням недели
- Посещения по часам
- Популярность абонементов

---

#### Безопасность
- Токен бота вынесен в переменные окружения (.env)
- Файл .env добавлен в .gitignore
- Конфиденциальные данные не хранятся в репозитории

---

#### Демонстрация функционала
1. Пользователь запускает бота
2. Покупает абонемент
3. Получает ID (аналог QR)
4. Подтверждает посещения
5. Система ведет учет и обновляет статус
6. Данные используются для построения аналитики
